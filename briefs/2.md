# MISSION BRIEF: LEVEL 3 (ADVANCED - THE CORE)

## Storyline
You have arrived at the reactor core. A custom background service is running on the system, listening on a hidden network socket. The service holds the advanced core flag. You need to inspect the running processes, identify the listening socket, and query the local service to obtain the final flag.

## Objectives
1. Inspect running processes using `ps` or network sockets using `netstat` / `ss`.
2. Locate the service listening on a local port (e.g. 127.0.0.1:9090).
3. Connect to the socket using `curl` or `nc` to retrieve the flag.

## Helpful Commands
- `ps aux` (List running processes)
- `ss -tlnp` or `netstat -tlnp` (List listening sockets with PID/Name)
- `curl http://127.0.0.1:<port>` (Send HTTP request)
- `nc 127.0.0.1 <port>` (Open TCP socket connection)
