# MISSION BRIEF: LEVEL 1 (BASE - GROUND ZERO)

## Storyline
Welcome to the Linux Dungeon, Operative. You have breached the outer boundary of the facility. To progress further, you need to explore the file system and find the authentication code left in a hidden directory.

## Objectives
1. Explore the file system using commands like `ls`, `cd`, `pwd`.
2. Locate the hidden directory under `/home/player/dungeon/base/`.
3. Read the file containing the level authentication flag and submit it.

## Helpful Commands
- `ls -la` (List all files including hidden ones)
- `cd <directory>` (Change directory)
- `cat <file>` (Print file content)
