# MISSION BRIEF: LEVEL 2 (INTERMEDIATE - THE PATH)

## Storyline
The security systems have locked the pathway leading to the main reactor core. You must locate a configuration file that contains the flag. However, it is owned by the root group and restricted. You need to search for setuid binaries or files with special permissions to read it.

## Objectives
1. Search the filesystem for specific files owned by root but readable or executable by the player.
2. Find the flag by using grep to filter files inside `/home/player/dungeon/intermediate/`.
3. Alternatively, find the SUID helper executable that can display the contents of the flag stored at `/root/1`.

## Helpful Commands
- `find / -perm -4000 -type f 2>/dev/null` (Find files with SUID permissions)
- `grep -r "FLAG{" /path` (Search recursively for pattern)
